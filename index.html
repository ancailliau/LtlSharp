LittleSharp

A library to check LTL formulae for satisfiability and validity.

You can download this project in either zip or tar formats. Binaries for the version 0.1 for Linux are available here.

You can also clone the project with Git by running: 

$ git clone git://github.com/ancailliau/LittleSharp

Or simply browsing source code.
